1) Enumerations - related group of constants
    > group values together, example:
        enum Directions {
            north, east, south, west
        }
        var direction = Directions.west;

        enum Damage {
            normalDamage = 1;
            doubleDamage = 2;
            quadDamage = 4
        }
        var damage = (int) Damage.quadDamage * 10; // have to cast enumeration

    > const int x = 1; // dont change the variable

4) Switch Statements
    > example:
        switch (playerCount)
        {
            case 1:
                message = "Ready Player One";
                break ;
            case 2:
                message = "Ready Player Two";
                break ;
            case default:
                message = "someone else";
                break ;
        }
